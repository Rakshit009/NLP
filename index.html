Preprocessing Library in PHP

Data cleaning code designed in Php

Use in project domains such as ML, NLP, AI to clean data

To use tokenizer download package from Nlp Tools or install using Composer
Commands to install composer and nlp tools package
$ sudo apt-get install composer
$ composer require nlp-tools/nlp-tools  // installing nlp tools in project folder
importing NLPTools in Our Program
use NlpTools\Tokenizers\WhitespaceAndPunctuationTokenizer;  // for using another tokenizer just replace with greentext
include('vendor/autoload.php');
Code to handle pre-processing
$replaced_emoji = $this -> emoji_replacer($review); //Emoji Replacer
$lower = strtolower($replaced_emoji); // Data Lowering
$stripped = trim(preg_replace('/\s+/', ' ', $lower)); // Space Stripping
$contraction_result = $this -> contraction_remover($stripped); // Contraction Handler
$polarity_result = $this -> polarity_replacer($contraction_result); // Polarity Checker
$punctuation_removed = preg_replace('/[^a-z]+/i', ' ', $polarity_result); // Punctuation Remover
$stop_words_removed = $this -> removeCommonWords($punctuation_removed); // Stop Words Remover
Creating and Applying Object on Natural text
$tokens = new WhitespaceAndPunctuationTokenizer(); // White space and Punctuation Tokenizer
$this -> tok = $tokens -> tokenize($stop_words_removed); // Creating Tokens
